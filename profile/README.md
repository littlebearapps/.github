I'm Nathan — this is where I build and share developer tools.

Everything here started as something I needed in my own workflows. MCP servers, Claude Code plugins, documentation generators — if I hit a wall often enough, I build something and open-source it.

**What's here:**

- **[PitchDocs](https://github.com/littlebearapps/pitchdocs)** — Give your AI coding agent the skills to scan your codebase and produce professional repo docs, user guides, changelogs, and context files
- **[Untether](https://github.com/littlebearapps/untether)** — A Telegram bridge for AI coding agents. I walk my dog for hours and run Claude Code across half a dozen projects from my phone — voice notes and all
- **[Outlook MCP](https://github.com/littlebearapps/outlook-mcp)** — Give Claude access to your Outlook email, calendar, and contacts via Microsoft Graph

All MIT-licensed. All privacy-first — your data stays on your machine.
